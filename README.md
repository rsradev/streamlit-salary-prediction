
Quickstart:

Download data: 
`sh get_data.sh`
if you're not on ARM system please rerun `ipython Prediction.ipynb` to recreate `steps.pkl`

Run project:
`pip install -r requirements.txt && streamlit run app.py`

Docker:
`sh build.sh && sh run.sh`
